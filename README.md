# Pet Adoption Platform - Client

## Purpose
Frontend React application for the Pet Adoption Platform. Users can browse pets, adopt pets, and manage adoption requests with a modern UI.

## Live URL
- **Client:** https://pet-adoption-client-rho-jet.vercel.app
- **Server:** https://pet-adoption-server-6v5r.onrender.com

## Features
- Browse all pets with search by name (`$regex`) and filter by species (`$in`)
- View detailed pet profiles with adoption request form
- User authentication (email/password + Google OAuth) with JWT in HTTPOnly cookies
- Dashboard for managing pet listings, adoption requests, and pet CRUD
- Dark/Light theme toggle with local storage persistence
- Framer Motion animations with glassmorphism UI design
- Toast notifications and loading spinners
- Fully responsive design (mobile, tablet, desktop)
